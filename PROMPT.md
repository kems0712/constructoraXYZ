**Actúa como un Desarrollador Web Full-Stack Senior, experto en React, Tailwind CSS, UI/UX y conversión (Lead Generation).**

**Contexto del Proyecto:**
Quiero construir desde cero la página web de una empresa de construcción. El objetivo principal es que la web sea profesional, clara, moderna, de carga muy rápida y que esté estratégicamente pensada para vender servicios y generar consultas. 

A continuación, te proporcionaré las especificaciones de la estructura. Tu trabajo es entregarme la arquitectura del proyecto, los comandos de instalación iniciales y el código completo de los componentes utilizando **React (con Functional Components y Hooks)** y **Tailwind CSS** para los estilos.

**Requerimientos de Estructura y Funcionalidad:**

1. **Hero Section (Inicio):**
   * Debe ser el primer impacto visual.
   * Mostrar de forma clara e inmediata a qué se dedica la empresa.
   * Destacar de forma resumida los servicios principales.
   * Incluir un botón de llamada a la acción (CTA) de contacto muy visible.

2. **Sección "Sobre Nosotros":**
   * Espacio para presentar a la empresa.
   * Resaltar la experiencia en el sector para transmitir confianza y autoridad.

3. **Sección de "Servicios":**
   * Un *grid* moderno donde se explique todo lo que hace la empresa de forma clara, ordenada y visualmente atractiva.

4. **Sección de "Proyectos Realizados" (Portafolio):**
   * Una vitrina que muestre el trabajo real de la empresa.
   * **Crucial:** Implementar un sistema de enrutamiento (por ejemplo, usando React Router) para que cada proyecto tenga su propia URL y página individual.
   * La vista individual de cada proyecto debe incluir: una galería de fotos, la descripción del proyecto y las características técnicas del trabajo realizado.

5. **Sección de "Testimonios":**
   * Una cuadrícula o carrusel con reseñas de clientes para generar prueba social.

6. **Integración de Contacto (WhatsApp):**
   * Implementar un botón o widget de chat flotante global que permita abrir la conversación de WhatsApp con un solo clic y enviar un mensaje predeterminado.

**Instrucciones de Salida:**
1. **Paso 1:** Proporciona los comandos exactos para inicializar este proyecto de React con Tailwind CSS (te sugiero usar Vite para mayor velocidad).
2. **Paso 2:** Diseña y muéstrame una estructura de carpetas lógica, limpia y escalable para este proyecto.
3. **Paso 3:** Escribe el código completo y modular para cada una de las páginas y componentes solicitados. Asegúrate de que el diseño sea totalmente *responsive* (Mobile First) utilizando las clases de Tailwind.
4. **Paso 4:** Utiliza datos de relleno (*mock data*) estructurados en arreglos o archivos JSON para los servicios, los proyectos y los testimonios. De esta forma, el componente quedará limpio y yo solo tendré que reemplazar la información real más adelante.
